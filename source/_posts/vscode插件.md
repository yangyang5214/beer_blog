### Insert Date String

ctrl + shift + i

### JSON Tools

alt + m    JSON minify.

ctrl + alt + m   JSON pretty.

### tencent-cloud-cos-upload-image

上传图片到 tencent cos

shift + p 直接上传剪贴板的图片

shift + o 上传本地图片
